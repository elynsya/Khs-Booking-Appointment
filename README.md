A full-stack web application developed as part of the Software Development for Information System course. This platform streamlines medical scheduling by providing an intuitive booking system for patients and a centralized management dashboard for healthcare providers.

---

##  Features

### For Patients
* **Dynamic Appointment Booking:** Seamlessly schedule, reschedule, or cancel clinic appointments in real-time.
* **Real-Time Availability:** View updated doctor schedules to prevent double-booking incidents.
* **User Authentication:** Secure signup and login portal powered by Firebase Authentication.

### For Medical Staff & Admin
* **Availability Dashboard:** A comprehensive, real-time interface to monitor upcoming daily appointments and manage patient queues[cite: 1].
* **Status Management:** Ability to approve, update, or clear scheduled slots instantly.

---

##  Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap
* **Backend & Database:** Firebase Firestore (Real-time Database)
* **Authentication:** Firebase Auth
* **Development Environment:** Visual Studio Code
* **Deployment/Testing:** VS Code Live Server Extension
